# 直播平台前端原型实现计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 用纯前端技术（Vue2 CDN + 原生 CSS）实现三端可交互原型页面，覆盖学生端、老师客户端、管理后台，使用 Mock 数据模拟所有交互，无需真实后端。

**Architecture:** 三个独立的单页应用，每端一个目录，使用 Vue2 CDN 引入，无需构建工具。所有数据用 JS 对象 Mock，页面间跳转用 hash router 实现。

**Tech Stack:** Vue2 (CDN), 原生 CSS, Mock JS 数据, 浏览器原生 API

```

│  [课程名称]  🔴 直播中  时长: 00:23:15  [结束直播]│
├──────────────────────┬───────────────────────┤
│                      │  在线学生 (12)         │
│   屏幕共享预览区       │  ─────────────────    │
│   （灰色占位/真实共享） │  学生列表（可禁言）     │
│  ┌────┐              │  ─────────────────    │
│  │摄像│ 画中画         │  聊天消息              │
│  └────┘              │  ─────────────────    │
├──────────────────────┤  输入框 + 发送          │
│ [共享屏幕▼] [摄像头🔴] │                       │
│ [麦克风🔴]  [虚拟背景] │  [全体禁言] [清空聊天]  │
└──────────────────────┴───────────────────────┘
```

功能：
- 屏幕共享：调用 `getDisplayMedia` 真实 API，选择桌面或应用
- 摄像头开关：调用 `getUserMedia`，画中画显示
- 麦克风开关：静音/取消静音
- 虚拟背景：点击弹出背景选择（3个预设背景图 + 模糊）
- 直播计时器：实时计时
- 学生列表：Mock 数据，每个学生有"禁言"按钮
- 全体禁言按钮：切换状态
- 聊天：发送/接收 Mock 消息
- 结束直播：确认弹窗后跳回 `#/courses`
